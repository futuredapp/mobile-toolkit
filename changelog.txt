  🛠 This is a maintenance release 1.4.2 - over a dozen long-standing bugs fixed!

    🍏 go-ios is now installed as a pinned, checksum-verified release binary (v1.2.0)
    Existing installations are offered the update automatically - no Go toolchain needed anymore

    💿 Developer disk images are now mounted automatically via "go-ios image auto"
    The old "open Xcode and wait" workaround is gone

    📱 Device names of all modern iPhones and iPads are now resolved correctly
    (via Xcode's device database with an online fallback - no more raw board codes)

    🔪 ikill now relaunches the app on the correct device when several are connected

    📜 alog -f now uses logcat --pid for reliable filtering on current Android versions

    🔊 atalkback works again on current TalkBack versions and turns off cleanly

    📹 arecord/isimulator recording fixes, aemulator setup fixes (JAVA_HOME, auth token)

    🔄 The update check no longer aborts the command you just ran

    ⭐️ Mobile Toolkit is now maintained by Futured (https://github.com/futuredapp)
    Built on the original work of Adam Svoboda - thanks, mate!
